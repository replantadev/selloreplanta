# Mis Plugins Personalizados de Replanta

Este repositorio contiene los plugins personalizados desarrollados para Replanta.net.

## Plugins incluidos:

- **replanta-republish-ai**: Plugin para republicar contenido con IA
- **selloreplanta-main**: Plugin principal del sello Replanta
- **truspilot-replanta**: Integración con Trustpilot
- **dominios-reseller**: Plugin para reseller de dominios
- **indice**: Plugin para índice de contenidos

## Instalación

1. Clona este repositorio en la carpeta `wp-content/plugins` de tu instalación de WordPress
2. Activa los plugins desde el panel de administración de WordPress

## Desarrollo

Para contribuir al desarrollo de estos plugins:

1. Haz fork del repositorio
2. Crea una rama para tu feature
3. Realiza tus cambios
4. Envía un pull request

## Producción

Este repositorio está sincronizado con el servidor de producción en replanta.net
